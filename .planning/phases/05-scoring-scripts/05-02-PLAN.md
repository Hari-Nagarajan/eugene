3. **src/tools/run_script.rs**: Create RunScriptTool.
   - Struct: `pub struct RunScriptTool { memory: Arc<Connection>, config: Arc<Config> }`
   - Constructor: `pub fn new(memory: Arc<Connection>, config: Arc<Config>) -> Self`
   - Args: `RunScriptArgs { name: String, timeout: Option<u64> }` -- timeout defaults to 60 if None
   - Output: `RunScriptResult { name: String, stdout: String, stderr: String, exit_code: i32, success: bool }` (Serialize)
   - NAME: "run_script"
   - definition(): JSON schema with name (required), timeout (optional integer). Description: "Execute a saved script by name. Fetches the script from the database, writes to a temp file, and executes it."
   - call():
     a. Fetch script: `get_script_by_name(&self.memory, args.name.clone()).await?`. If None, return `ToolError::ExecutionFailed(format!("Script not found: {}", args.name))`
     b. Create temp file: `tempfile::Builder::new().prefix("eugene-script-").suffix(suffix).tempfile()` where suffix is ".sh" for bash, ".py" for python. Map io errors to ToolError::ExecutionFailed.
     c. Write code to temp file: `tmp.write_all(script.code.as_bytes())`. Import `std::io::Write`.
     d. Execute via interpreter (NOT direct execution -- avoids permission issues per RESEARCH.md Pitfall 4):
        - bash: `tokio::process::Command::new("bash").arg(tmp.path())`
        - python: `tokio::process::Command::new("python3").arg(tmp.path())`
     e. Apply timeout using `tokio::time::timeout(Duration::from_secs(timeout), child.wait_with_output())`. On timeout, return ToolError::Timeout.
     f. Update usage: `update_script_usage(&self.memory, script.id).await` (ignore errors -- entropy-goblin error-as-value pattern for non-critical updates)
     g. Return RunScriptResult with stdout/stderr as String::from_utf8_lossy, exit code, success flag.
     h. Temp file auto-cleaned on drop when `tmp` goes out of scope.
